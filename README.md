A professional website for Roots & Technique, specializing in premium afro locs installation, retwists, repairs, and maintenance.

## 🌟 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Professional Layout** - Modern, clean design showcasing services and transformations
- **Easy Navigation** - Smooth scrolling between sections
- **WhatsApp Integration** - Direct booking through WhatsApp
- **Gallery Section** - Showcase your work and transformations
- **Service Cards** - Highlight your core services

## 📁 Files

- `index.html` - Main website structure
- `style.css` - Custom styling with gold and dark theme
- `images/` - Directory for gallery images (create and add your photos here)

## 🚀 Setup Instructions

### 1. Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Select **main** branch and save
4. Your site will be available at: `https://Roots-afro-Lox.github.io/Roots-Technique-/`

### 2. Add Your Gallery Images
1. Create a folder named `images` in your repository
2. Upload your gallery photos:
   - `gallery1.jpg`
   - `gallery2.jpg`
   - `gallery3.jpg`
3. (Optional) Add more images and update the HTML accordingly

### 3. Update Contact Information
Replace the WhatsApp number in the HTML:
- Current: `https://wa.me/27672558552
- Update to your actual number in `index.html`
